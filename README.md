## 📌 Sobre o Projeto

Este é meu projeto de portfólio pessoal, desenvolvido com foco em apresentar minhas habilidades como desenvolvedor backend.

O objetivo é demonstrar conhecimentos em desenvolvimento web, organização de código e integração com tecnologias modernas utilizadas no mercado.

---

## 🧠 Sobre Mim

Sou Michel Pereira da Cunha, desenvolvedor backend especializado em Java e Spring Boot.

Tenho experiência na construção de APIs RESTful, desenvolvimento de sistemas CRUD, conteinerização com Docker e integração com bancos de dados MySQL, além de conhecimentos em AWS.

Ajudo pequenos negócios e profissionais a transformarem boas ideias em soluções digitais eficientes, escaláveis e seguras.

## 🌐 Como Executar

1. Clone o repositório:

```bash
git clone git@github.com:Michel-P-C/portifolio.git
```

2. Acesse a pasta:

```bash
cd portifolio
```

3. Abra o arquivo:

```bash
index.html
```

Ou utilize a extensão **Live Server** no VS Code.
